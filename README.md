# HCV (http comic view)
#### Set of file to deploy image viewer on server

## Prerequisite
- Server set up with php

## How to use
1. Download zip on the release page
2. Extract zip
3. Deploy on server
4. Set server direct to "ht" folder
5. Deploy image in "img" folder

------------
###### Test on Apache server 2.4 + PHP 7.4.10
